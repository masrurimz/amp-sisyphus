# Sisyphus Plans

This directory contains work plans created by Prometheus.

## Naming Convention

`YYYY-MM-DD-[slug].md`

Example: `2026-01-11-auth-refactor.md`

## Plan Lifecycle

1. **Created** - Prometheus interviews user, creates plan
2. **Reviewed** - Momus critically reviews
3. **Approved** - Ready for implementation
4. **In Progress** - Sisyphus orchestrator executing
5. **Completed** - All tasks done, verified

## Plan Template

See `_template.md` for the standard plan structure.
